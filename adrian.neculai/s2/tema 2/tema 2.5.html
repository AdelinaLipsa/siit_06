An example using non-semantic mark-up

This is one of two pages created to demostrate the difference between pages using semantic HTML mark-up and those that don't.

What you see isn't always what you get.

If you compare this page with the one useing semantic mark-up, you win't see significant differences in oyur browser.

But what's on the inside can make a world of difference for accessibility, search engine optimization, and workflow. Read the accompanying blog entry to learn more.
